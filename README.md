# AI Image & Video Generator

This is a web-based AI application that converts text prompts into images and short videos using cloud-based AI APIs. The project is designed to be mobile-first and can be created and managed using an Android phone.

## Features
- Text to Image generation
- Text to Video generation
- Simple and clean user interface
- Mobile-friendly design
- API-based AI integration
- Monetization-ready structure

## How It Works
1. User enters a text prompt
2. The prompt is sent to the backend server
3. AI model generates an image or video
4. Result is displayed to the user

## Technologies Used
- HTML, CSS, JavaScript
- Node.js (Backend)
- Cloud AI APIs (Image & Video generation)
- GitHub Pages (Frontend hosting)

## Setup
1. Clone the repository
2. Install dependencies
3. Add your API keys in a `.env` file
4. Run the backend server
5. Open the website in a browser

- Subscription plans
